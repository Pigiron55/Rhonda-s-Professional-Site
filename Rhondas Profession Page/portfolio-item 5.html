Using Javascript to create a Password Generator

Generating Secure Passwords

I used Javascript to create a random password generator that can generate randomized passwords using Letters, Numbers and Symbols.

>>Password Generator<<

You can choose the length of the password from 8-20 characters. You can also choose any combination of letters, numbers and symbols.

The generator creates two unique passwords derived from the specified imputs and allows you to copy either to your clipboard to paste into the password creation field.
